# 🛒 Cloth Rental - Frontend

This is the client-side application for the **Cloth Rental E-Commerce System**, built using **React.js** and styled with **Tailwind CSS**.

## 🚀 Getting Started

### Prerequisites
Ensure you have **Node.js** and **NPM** installed.

### Installation
1. Navigate to this directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
   The app will be available at `http://localhost:3000`.

## 🛠 Key Features
*   **Dynamic Product Discovery** – Filter by category and size.
*   **Rental Date Picker** – Integrated calendar for booking selection.
*   **Responsive UI** – Fully optimized for all device sizes.
*   **State Management** – Utilizes React Context for a smooth cart experience.

## 📁 Directory Structure
*   `/src/Components` – Reusable UI components.
*   `/src/Pages` – Main application pages.
*   `/src/Context` – State management logic.
*   `/src/Assets` – Static images and icons.
